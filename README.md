# MyBlog - PHP & MySQL Blog Application

A fully functional blog application built with PHP and MySQL as part of my Web Developer Internship at ApexPlanet Software Pvt. Ltd.

## 🚀 Features

- User Registration and Login
- Secure Password Hashing
- Session Management
- Create, Read, Update, Delete (CRUD) Posts
- Search Posts by Title
- Pagination Support
- Role Based Access Control (Admin & Editor)
- Prepared Statements to prevent SQL Injection
- Form Validation
- Professional Green themed UI with Bootstrap 5
- Custom 404 Error Page

## 🛠️ Technologies Used

- PHP
- MySQL
- Bootstrap 5
- XAMPP
- Git & GitHub
